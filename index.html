React-Native

发表于 2017-05-18

基础

Flexbox 布局

Layout with Flexbox

  • Flexbox 可以指定一个 Component 子组的布局，用于适配不同尺寸的屏幕。
  • Flexbox 在 React Native 中的工作方式与在网络中的 CSS 相同，但其默认值不同， flexDirection 默认是 column 不是 row，flex 的参数仅支持单个数字。
Flex Direction
  • 使用 flexDirection 来确定其布局的主轴，其 children 应该是 row（水平） 或者 column， 默认是 column(垂直).
Justify Content
  • 将 justifyContent 添加到 component 的 style 中，将决定其 children 沿主轴上的分布，其可选项为：flex-start, center, flex-end, space-around, and space-between.
Align Items
  • 将 alignItems 添加到 component 的 style 中，将决定其 children 沿副主轴的对齐方式（若 主轴是 row 则辅助轴是 column)，其可选项为：flex-start, center, flex-end, and stretch.

输入框的使用(TextInput)

  • TextInput 是一个基本的 Component 允许用户输入文本，当输入内容改变的时候会调用 onChangeText 函数，以及当文本内容被提交的手会调用 onSubmitEditing 函数。

扩展名区分平台

Platform 区分

  • Platform 模块是 RN 提供的内部模块，主要用于平台区分。

主要的使用方法为Platform.select, 框架会自动区分当前平台，进行组件选择

MBProgressHUD 源码

发表于 2017-05-11

是什么？

  • MBProgressHUD 是一个iOS插件类，在后台线程中完成工作时，显示带有指示符或标签的半透明 HUD。 HUD意味着替代未记录的私有 UIKit UIProgressHUD 以及一些其他功能。

使用要求

  • MBProgressHUD 适用于 iOS 6+，并要求 ARC 进行构建。它依赖于以下 Apple 框架：
    • Foundation.framework
    • UIKit.framework
    • CoreGraphics.framework

添加 MBProgressHUD 到你的项目

  • CocoaPods（推荐使用）

    1. 将 MBProgressHUD 的 pod 条目添加到你的 Podfile：pod ‘MBProgressHUD’, ‘~> 1.0.0’
    2. 通过运行安装 pod(s) : pod install
    3. 使用 MBProgressHUD 的地方需要加入： #import “MBProgressHUD.h”.

  • 源文件

    1. 直接将 MBProgressHUD.h 和 MBProgressHUD.m 源文件添加到项目中
    2. 包含 MBProgressHUD 的地方需要加入： #import “MBProgressHUD.h”.

  • 静态库

    1. 下载最新的代码版本，生成 .a 静态库
    2. 将 MBProgressHUD 添加到目标依赖关系(Target Dependencies list)列表中.

使用

  • 使用 MBProgressHUD 的推荐方法是在主线程上进行设置，然后将要执行的任务旋转到新线程上
  • 你可以在任何的 View 或者 window 中使用 HUD，但是，避免将 HUD 添加到具有复杂视图的某些 UIKit 视图(如 UITableView 或 UICollectionView )中，可能会打破 HUD 显示。
  • HUD 进度的使用：
  • 您也可以使用 NSProgress 对象，当通过该对象报告进度时，MBProgressHUD 将自动更新。
  • 如果在主线程上执行一个长时间的任务，你需要延迟执行，从而给主线程足够的时间绘制 HUD

许可

  • 该代码根据MIT许可证的条款和条件分发。

HTML学习笔记二

发表于 2017-04-08

HTML 基础

表格的应用

  1. 创建表格：
    • 表格的基本构成：<table>(表格标记)、<tr>(行标记)、td（单元格标记）；
    • 表格的标题：<caption></caption>；
    • 表格的表头：<th> 一般位于表格的第一行；
  2. 表格的基本属性：
    • width、height 其值为浏览器的百分比或固定像素数；
    • align： 表格的对齐方式；
    • border: 表格边框的宽度；
    • bordercolor：表格边框的颜色；

层标记-Div

  1. <div>标签：style 属性用于设置对象的内嵌样式

编辑表单

  1. 使用表单标签：<form>

    • action：处理动作，真正处理表单的数据脚本或程序在 action 属性里，这个值可以是程序或脚本的一个完整 URL；
    • name：表单名称（不能包含特殊字符和空格）;
    • method：用于定义处理程序从表单中获取信息的方式，它决定了表单中已收集的数据是用什么方法发送到服务器的。传送方式只有两种，即 get 或 post ；
    • enctype：编码方式，text/plain(以纯文本的形式传送)、application/x-www-form-urlencoded(默认的编码形式)、multipart/form-data(MIME编码，上传文件的表单必须选择该项)；

    • target：目标显示方式，_blank/_parent/_self/_top;

  2. 文字字段:text

    <input type="text" name="userName" size="20" placeholder="text" value="default value" maxlength="40">

  3. 密码域：password

    <input type="password" name="userName" size="20" value="123456" maxlength="8">

  4. 单选按钮：radio

    checked 属性表示这一单选按钮默认被选中，只能设置一项单选按钮控件为 checked，value 设置用户选中该选项后，传送到处理程序中的值。

  5. 复选框：checkbox

  6. 普通按钮：button

    <input type="button" value="关闭当前窗口" name="close" onclick="window.close()">

  7. 提交按钮：submit

    <input type="submit" name="submit" value="提交表单">

  8. 重置按钮：reset

    <input type="reset" name="reset" value="重置表单">

    value 的值用于设置按钮上显示的文字

  9. 图像域：image

    <input type="image" name="image1" src="http://pic.58pic.com/58pic/12/06/87/43R58PICm3U.jpg" size="40" height="40">

  10. 隐藏域：hidden

    <input type="hidden" name="form" value="invest">

    表单中的隐藏域主要用来传递一些参数，而这些参数不需要再页面中显示，当用户提交表单的时候，隐藏域中的内容会一起提交给处理程序。

  11. 文件域：file
    请上传你的照片:<input type="file" name="file"><br>

  12. 列表、菜单标记

    • name：菜单和列表的名称；
    • size: 显示的选项数目；
    • multiple：列表中的项目多选；
    • value：选项值；
    • selected：默认选项

  13. 文本域标记 textarea（在留言本中最为常见）

    • name：文本域的名称；
    • rows：文本域的行数；
    • cols：文本域的列数；
    • value：文本域的默认值；（测试时不好使）

    <textarea name="liuyan" rows="5" cols="40" placeholder="请留言...">默认展示的内容</textarea>

多媒体页面

  1. 滚动文字标签：marquee

    • direction：滚动方向，分别为 up、down、left、right，默认为 left；
    • behavior：滚动方式，scroll（循环滚动，默认效果）、slide（只滚动一次就停止）、alternate（来回交替进行滚动）；
    • scrollamount：滚动速度属性；
    • scrolldelay：滚动延迟属性，时间间隔单位是毫秒，滚动两步之间的时间间隔；
    • loop：循环属性，用于设定循环次数；
    • width、height：滚动范围属性；
    • bgcolor：滚动背景颜色属性；

    • hspace、vspace：滚动空间属性，在滚动文字的四周，可以设置水平和垂直范围（单位均为像素）；

      <marquee behavior="alternate" direction="down" loop="100" scrollamount="3" width="500" height="100" bgcolor="#FF0000">奔跑吧第一行</marquee>

  2. 添加背景音乐：<bgsound src="背景音乐地址" loop="次数">

    • loop：设置循环播放次数，如果希望无限次播放背景音乐，可将 loop 的值设置为 true；

  3. 添加多媒体文件：embed

    • autostart：设置自动运行，true、false；
    • loop：设置媒体文件的循环播放，true、false；

    • hidden：隐藏面板，true、false；

  4. 除 AVI 媒体文件外，在网页中还可以嵌入 Flash、MPEG 等类型的媒体文件，方法与上相同；

HTML学习笔记一

发表于 2017-04-05

HTML 基础

HTML 的基本结构

  1. 一个 HTML 文件是由一系列的元素和标签组成的，HITML 的标签分为单独标签和成对标签两种。在每个 HTML 标签中不区分大小写。
  2. 当用一组 HTML 标签将一段文字包含在中间时，这段文字与包含文字的 HTML 标签被称为一个元素。
  3. 文件的开始标签 <html> 文件的全部内容 </html>
  4. 文件的头部标签 <head> </head> 主要包含文件的标题、编码方式、URL等信息，如果不提供，可省略。
  5. 文件主题标签 <body></body> 包含该网页的主体内容。

HTML 文件基本标记

  1. 标题标记 <title>...</title> 标记的内部就是标题的内容。

  2. 元信息标记 <meta ...> 提供的信息，不显示在页面中，一般用于定义页面信息的名称、关键字、作者等， meta 元素的属性有两种：name 和 http-equiv：

    1. 设置页面关键字，便于帮助搜索引擎对该网页进行查找和分类：<meta name="keyname" content="关键字1,关键字2">
    2. 设置页面描述：<meta name="description" content="对页面的描述语言">
    3. 设置编辑工具: <meta name="generator" content="sublime">
    4. 设定作者信息：<meta name="author" content="zhangsan">
    5. 限定搜索方式：<meta name="robots" content="All/Index/Nofollow/Noindex/None"> (All: 表示能搜索当前网页及其链接的网页；Index：表示能搜索当前网页；Nofollow：表示不能搜索当前网页链接的网页；Noindex：表示不能搜索当前页；None：表示都不能搜索)。
    6. 设置网页文字及语言：
      <meta http-equiv="Content-Type" content="text/html" charset="utf-8">
      <meta http-equiv="Content-Language" content="en">。
    7. 设置网页的定时跳转：<meta http-equiv="refresh" content="跳转时间;url=链接地址"> （跳转时间是以秒为单位）。
    8. 设置有效期限：<meta http-equiv="expires" content="Wed,14 september 2017 23:14:00 GMT"> 到期的时间必须是 GMT 格式时间。
    9. 禁止从缓存中调用：
      <meta http-equiv="cache-control" content="no-cache">
      <meta http-equiv="pragma" content="no-cache">
    10. 删除过期的 cookie：
      <meta http-equiv="set-cookie" content="Wed,14 september 2017 23:20:00 GMT">
    11. 强制打开新的窗口：（可以防止自己的网页被当做一个frame页调用）
      <meta http-equiv="windows-target" content="_top">
      在该语法中，windows-target 表示新网页的打开方式，而_top表示打开的是一个独立页面。
    12. 设置网页的过度效果：
      <meta http-equiv="page-exist/page-enter(过渡事件)" content="revealtrans(duration=3（时间）,transition=21（方式）)">

  3. 基底网址标志<base> ：
    <base href="链接地址" target="新窗口的打开方式">

    • _parent:在上一级窗口中打开，一般常用在分针的框架页中
    • _blank: 在新窗口中打开
    • _self: 在同一窗口打开，可省略
    • _top: 在浏览器的真嘎窗口打开，忽略任何框架
  4. 页面的主体标记<body>:
    • text: 设定页面的文字颜色
    • bgcolor：设定页面的背景颜色
    • backgroud：设定页面的背景图片
    • bgproperties:设定页面的背景图片为固定，不随页面的滚动而滚动（值设置为 fixed 图片会固定在页面上）
    • link：设定页面默认的链接颜色
    • alink：设定鼠标正在单击时的链接颜色
    • vlink：设定访问过后的链接颜色
    • topmargin：设定页面的上边距
    • leftmargin：设定页面的左边距
  5. 页面的注释标记： <!--注释的文字-->

  2. 除了绝对路径和相对路径外，还有一种称为根目录，根目录常常在大规模站点需要放置在几个服务器上，或者一个服务器上放置多个站点时使用，用法只需要以”/“开始表示根目录，之后是文件所在的目录名和文件名，如 /mr/5-1.html;
  3. 超链接的建立：
    <a href="链接地址" target="目标窗口的打开方式">链接元素</a>
    • href: 目标的链接地址
    • target: 目标窗口的打开方式(_parent、_blank、_self、_top)
  4. 书签链接：
    • 建立书签:<a name="书签名称">文字</a> (数千名称就是设置连接后要跳转的位置)
    • 链接到同一页面的书签：<a href="#书签名称"></a>
    • 链接到不同页面的书签: <a href="链接的文件地址#书签名称"></a>
